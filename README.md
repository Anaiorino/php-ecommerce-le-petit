# Le Petit Papelaria | Trabalho 4º módulo ADS

E-commerce completo desenvolvido em PHP + MySQL.

## Funcionalidades
- Cadastro e Login
- Carrinho persistente
- Checkout PIX
- Painel Admin
- Upload de produtos
- Gestão de pedidos

## Tecnologias
PHP, MySQL, HTML, CSS, JavaScript

## Objetivo
Projeto criado para portfólio profissional focado em vagas de desenvolvimento web.
